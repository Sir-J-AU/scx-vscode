BATCHES

List Batches

List batches belonging to the current organization. Results are ordered by creation date descending.

GET/v1/batches

Authorization

Authorizationstringheaderrequired

Bearer token — your API key. Example: Bearer sk-...

Response

object"list"required

Object type — always "list".

dataobject[]required

Array of batch objects.

first_idstringrequired

The ID of the first batch in the list.

last_idstringrequired

The ID of the last batch in the list.

has_morebooleanrequired

Whether there are more results available.

Request

import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "your-scx-api-key",
  baseURL: "https://api.scx.ai/v1",
});

const list = await client.batches.list({ limit: 20 });

for (const batch of list.data) {
  console.log(batch.id, batch.status);
}

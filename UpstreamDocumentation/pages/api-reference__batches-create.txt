POST /v1/batches Authorization Authorization string header required Bearer token — your API key. Example: Bearer sk-... Request body multipart/form-data input_file_id string required The ID of an uploaded file that contains requests for the batch. Must have purpose &quot;batch&quot;. endpoint &quot;/v1/chat/completions&quot; | &quot;/v1/embeddings&quot; | &quot;/v1/responses&quot; required The endpoint to use for all requests in the batch. /v1/chat/completions /v1/embeddings /v1/responses completion_window &quot;24h&quot; The time frame within which the batch should be processed. Currently only &quot;24h&quot; is supported. 24h metadata object Optional metadata as key-value pairs. Maximum 16 pairs. Response id string required The batch ID (e.g. &quot;batch_abc123&quot;). object &quot;batch&quot; required Object type — always &quot;batch&quot;. endpoint string required The API endpoint used for requests in this batch. errors any required Validation or processing errors, if any. input_file_id string required The ID of the input file for this batch. completion_window string required The time window for batch completion. status &quot;validating&quot; | &quot;failed&quot; | &quot;in_progress&quot; | &quot;finalizing&quot; | &quot;completed&quot; | &quot;expired&quot; | &quot;cancelling&quot; | &quot;cancelled&quot; required The current status of the batch. validating failed in_progress finalizing completed expired cancelling cancelled output_file_id string required The ID of the output file with successful results. error_file_id string required The ID of the file containing error results. created_at integer required Unix timestamp of when the batch was created. in_progress_at integer required Unix timestamp of when processing started. expires_at integer required Unix timestamp of when the batch expires. finalizing_at integer required Unix timestamp of when finalization started. completed_at integer required Unix timestamp of when the batch completed. failed_at integer required Unix timestamp of when the batch failed. expired_at integer required Unix timestamp of when the batch expired. cancelling_at integer required Unix timestamp of when cancellation started. cancelled_at integer required Unix timestamp of when the batch was cancelled. request_counts object required Request processing counts. metadata object required User-provided metadata key-value pairs.

At least one of query or vector must be provided. POST /v1/vector-stores/{store_id}/search Authorization Authorization string header required Bearer token — your API key. Example: Bearer sk-... Request body application/json query string Text query to search for. At least one of query or vector is required. vector number[] Raw embedding vector to search with. At least one of query or vector is required. limit integer default: 10 Maximum number of results to return (1-100, default 10). scoreThreshold number Minimum similarity score threshold for results. Response data object[] required Array of search results ordered by relevance.

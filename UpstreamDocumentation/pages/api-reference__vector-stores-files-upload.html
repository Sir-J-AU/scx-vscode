VECTOR-STORES

Upload a File to a Vector Store

Upload a file to a vector store for embedding and indexing. The file is processed asynchronously after upload.

POST/v1/vector-stores/{store_id}/files

Authorization

Authorizationstringheaderrequired

Bearer token — your API key. Example: Bearer sk-...

Request body

multipart/form-data
fileanyrequired

The file to upload. Sent as multipart/form-data.

Response

idstringrequired

Unique identifier for the file.

vectorStoreIdstringrequired

ID of the parent vector store.

filenamestringrequired

Original filename.

contentTypestringrequired

MIME type of the file.

sizeBytesintegerrequired

File size in bytes.

objectKeystringrequired

Storage object key.

chunkCountintegerrequired

Number of chunks the file was split into.

status"pending" | "processing" | "completed" | "failed"required

Processing status of the file.

pendingprocessingcompletedfailed
errorstringrequired

Error message if processing failed.

createdAtstringrequired

ISO 8601 timestamp of creation.

updatedAtstringrequired

ISO 8601 timestamp of last update.

Request

const formData = new FormData();
formData.append("file", fs.createReadStream("document.pdf"));

const response = await fetch("https://api.scx.ai/v1/vector-stores/a1b2c3d4-e5f6-7890-abcd-ef1234567890/files", {
  method: "POST",
  headers: {
    "Authorization": "Bearer your-scx-api-key",
  },
  body: formData,
});

const file = await response.json();
console.log(file);

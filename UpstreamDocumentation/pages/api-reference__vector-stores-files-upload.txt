The file is processed asynchronously after upload. POST /v1/vector-stores/{store_id}/files Authorization Authorization string header required Bearer token — your API key. Example: Bearer sk-... Request body multipart/form-data file any required The file to upload. Sent as multipart/form-data. Response id string required Unique identifier for the file. vectorStoreId string required ID of the parent vector store. filename string required Original filename. contentType string required MIME type of the file. sizeBytes integer required File size in bytes. objectKey string required Storage object key. chunkCount integer required Number of chunks the file was split into. status &quot;pending&quot; | &quot;processing&quot; | &quot;completed&quot; | &quot;failed&quot; required Processing status of the file. pending processing completed failed error string required Error message if processing failed. createdAt string required ISO 8601 timestamp of creation. updatedAt string required ISO 8601 timestamp of last update.

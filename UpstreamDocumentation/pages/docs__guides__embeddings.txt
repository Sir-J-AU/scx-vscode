These embeddings are commonly used in applications such as recommendation systems, classification, and search. The Mistral embedding model powers our system, providing high-quality vector representations for various applications. For setup and integration: Get an API key from the API keys and URLs page. Generating embeddings The SCX.ai Embeddings API is designed for seamless integration with existing applications and follows OpenAI&#x27;s embeddings API structure, simplifying migration and adoption. The model takes in text as input and outputs a vector of floating-point numbers, which can be used for tasks like similarity searches. Generating a single embedding To generate an embedding for a single input text, use client.embeddings.create() with the model name and input string.

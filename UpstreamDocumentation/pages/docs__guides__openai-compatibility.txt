Download the library Run the command below to download the library. python pip install openai pip install openai Use SCX.ai APIs with OpenAI client libraries Configuring your OpenAI client libraries to use SCX.ai inference APIs is as simple as setting two values: the base_url and your api_key , as shown below. Note Don’t have a SCX.ai API key? Get yours from the API keys and URLs page. python from openai import OpenAI client = OpenAI ( base_url = &quot;https://api.scx.ai/v1&quot; , api_key = &quot;your-scx-api-key&quot; ) from openai import OpenAI client = OpenAI ( base_url = &quot;https://api.scx.ai/v1&quot; , api_key = &quot;your-scx-api-key&quot; ) Now you can make an API request to a model and choose how to receive your output. When calculating metrics like tokens per second or time per output token, ensure that you account for all tokens in each chunk. Currently unsupported OpenAI features The following features are not yet supported and will be ignored: logprobs top_logprobs n presence_penalty frequency_penalty logit_bias seed Feature differences temperature : The SCX.ai API supports a value between 0 and 1 whereas OpenAI supports values between 0 and 2. SCX.ai API features not supported by OpenAI clients The SCX.ai API supports the top_k parameter, which is not supported by the OpenAI client libraries.
